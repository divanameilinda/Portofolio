Tutorial Membuat Tabel

Perlu diperhatikan ketika membuat suatu tabel harus memahami beberapa istilah atau singkatan yang ada didalam syntax/code programnya, seperti :

  • tag <th> artinya table head (biasanya dikenal dengan judul di header tabel).
  • tag <tr> artinya table row (biasanya dikenal dengan baris di tabel).
  • tag <td> artinya table data (biasanya dikenal dengan kolom di tabel).

    • Berikut adalah langkah-langkah untuk membuat tabel sederhana:

    1. Gunakan tag <table> untuk membuat tabel.
    2. Tambahkan header tabel menggunakan tag <thead>.
    3. Masukkan data ke dalam tabel menggunakan tag <tbody>.
    4. Gunakan tag <tr> untuk setiap baris dan <th> untuk header serta <td> untuk data.

    Contoh syntax membuat tabel sederhana:

    <table>  -----> membuat tabelnya
  <caption>Contoh Tabel Sederhana</caption> ----->judul/caption
  <thead> ----->header
    <tr> ----->baris pertama untuk header
      <th>NIM</th> ----->nama kolom 1 : NIM
      <th>NAMA</th> ----->nama kolom 2 : NAMA
      <th>IPK (Semester 1-8)</th> ----->nama kolom 3 : IPK (Semester 1-8)
      <th>JURUSAN</th> ----->nama kolom 4 : Jurusan
      <th>FAKULTAS</th> ----->nama kolom 5 : Fakultas
    </tr> ----->tutup baris pertama
  </thead> ----->tutup header
  <tbody> ----->masuk data ke tabel
    <tr> ----->baris 1 untuk data di tabel
      <td>5220311048</td>
      <td>Divana Meilinda Putri</td>
      <td>3,89</td>
      <td>SISTEM INFORMASI</td>
      <td>SAINS & TEKNOLOGI</td>
    </tr> ----->ending/tutup baris 1 untuk data di tabel
    <tr> ----->baris 2 untuk data di tabel
      <td>5220311001</td>
      <td>Gyan Ferdiansyah Gunawan</td>
      <td>3,99</td>
      <td>PENDIDIKAN EKONOMI</td>
      <td>EKONOMI</td>
    </tr> ----->ending baris 2 untuk data di tabel
    <tr> ----->baris 3 untuk data di tabel
      <td>5220311077</td>
      <td>Melody Humairra</td>
      <td>3,50</td>
      <td>KEDOKTERAN GIGI</td>
      <td>KEDOKTERAN</td>
    </tr> ----->ending baris 3 untuk data di tabel
    <tr> ----->baris 4 untuk data di tabel
      <td>5220311077</td>
      <td>Syaira Mahreen</td>
      <td>3,51</td>
      <td>PENDIDIKAN MATEMATIKA</td>
      <td>MATEMATIKA</td>
    </tr> ----->ending baris 4 untuk data di tabel
  </tbody> ----->end body di tabel
</table>----->end table
