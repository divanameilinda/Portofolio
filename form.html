Tutorial Membuat Form

Perlu diperhatikan ketika membuat suatu form harus memahami beberapa istilah atau singkatan yang ada di dalam syntax/code programnya, seperti:

  • tag <form> artinya form (digunakan untuk mengelompokkan input dan kontrol lain).
  • tag <input> artinya input field (digunakan untuk menerima input dari pengguna).
  • tag <label> artinya label (memberikan deskripsi pada input field).
  • atribut action digunakan untuk menentukan URL untuk mengirim data.
  • atribut method yang menunjukkan cara pengiriman data (GET ketika mengambil data atau POST ketika mengirimkan data).
  • tag <button> untuk tombol yang dapat digunakan untuk mengirim atau mereset form.

Berikut adalah langkah-langkah untuk membuat form sederhana:
  1. Gunakan tag <form> untuk membuat form.
  2. Tambahkan input field menggunakan tag <input>.
  3. Gunakan tag <label> untuk memberikan label pada input.
  4. Atur atribut action dan method pada tag <form>.

Contoh syntax membuat form sederhana:

<form action="submit.php" method="POST">  -----> membuat form
  <label for="username">Username:</label>  -----> label untuk input username
  <input type="text" id="username" name="username" required>  -----> input field username
  <label for="password">Password:</label>  -----> label untuk input password
  <input type="password" id="password" name="password" required>  -----> input field password
  <input type="submit" value="Login">  -----> tombol submit
  <input type="reset" value="Reset">  -----> tombol reset
</form>  -----> end form
